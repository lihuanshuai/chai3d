Welcome to CHAI3D.

CHAI 3D is an open source, freely available set of C++ libraries for computer haptics, visualization and interactive real-time simulation.

I mainly improve make process and build a Python binding.

Installation

$ git clone https://github.com/lihuanshuai/chai3d.git
$ cd chai3d
$ git checkout libchai3d
$ make
$ git checkout python-build
$ python setup.py install
